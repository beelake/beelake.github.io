> ### # 如何添加友链
>
> 请首先添加我的友链，然后在**这个页面**下方的评论区里申请，申请格式：
>
> ```
> YAML
> - name: 
>   avatar: 
>   link: 
>   descr: 
> ```
>
> 为了方便博主添加友链，请遵从这个格式，谢谢！



> ### # 注意事项
>
> 1. 任何网址都可添加友链，包括但不限于博客，各类网站的主页，只要没有违法内容，都可以申请添加！
> 2. 无论您是大佬，神仙，还是萌新，蒟蒻，抑或是文化课选手，都来者不拒，欢迎互换友链！
> 3. 既然是互换友链，那么希望您能遵守承诺，如果您撤下了我的友链，那么相应的，您的友链也会被撤下，如果有特殊情况，可以在评论里留言。
> 4. 最后，感谢您能访问我的小站
